Class **Torpedo** provides some information about torpedoes.

You can get object of this type by function

    torpedoes = battle.getAllTorpedoes()

### Athributes:

- uiId -id of the torpedo iu element, int
- ownerId - int
- ownerTeamId - int
- ownerType - str
- shotId - int
- type - str
- position - Math.Vector3
